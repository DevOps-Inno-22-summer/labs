# IU DevOps Lab 1

##  Requirements
- Python 3.9+
- pip 22+

## Installation

Clone the repository

```bash
git clone https://github.com/behouba/devOpsLab

cd devOpsLab/app_python
```

Create and activate a virtual environement

```bash
python3 -m venv env

. ./env/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the app

```bash
python3 main.py
```


## Run tests

```bash
pytest .
```
