## Moscow local time

The web application that shows Moscow local time.

### Run

```sh
# Go to the app folder
cd app_python

# Create virtual environment
python3 -m venv env

# Open virtual environment
. ./env/bin/activate

# Upgrade pip
pip install -U pip

# Install requirements
pip install -r requirements.txt

# Run the app
python -m flask run
```
